# gitbook

## 我为什么选择gitbook

以前折腾过hexo，也折腾过gitbook本地版，这类工具相同的特点就是，编辑完成后需要本地构建出HTML静态文件，然后发布。 最挫的方法是手动构建和发布，改进思路是使用Travi自动构建和发布。

## 使用技巧

* 利用Typora来插图
* 可以增加文章来源`source`和文章在线编辑`edit-online`功能。
